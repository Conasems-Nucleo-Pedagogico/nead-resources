💡 Características da Animação Linear:
  • Os cards aparecem um após o outro
  • Movimento suave da esquerda para a direita
  • Timing controlado para melhor visualização
  • Ideal para apresentar sequências de informações

🛠️ Como Implementar

CSS Necessário:

.card-animacao {
    opacity: 0;
    transform: translateX(-20px);
    transition: all 0.6s ease;
}

.card-animacao.visivel {
    opacity: 1;
    transform: translateX(0);
}

JavaScript para Animação:

function animarCards() {
    const cards = document.querySelectorAll('.card-animacao');
    cards.forEach((card, index) => {
        setTimeout(() => {
            card.classList.add('visivel');
        }, index * 200); // Delay de 200ms entre cards
    });
}
⚠️ Dicas Importantes:
  • Use delays apropriados para não sobrecarregar o usuário
  • Teste em diferentes dispositivos
  • Considere preferências de movimento reduzido
  • Mantenha as animações suaves e profissionais
